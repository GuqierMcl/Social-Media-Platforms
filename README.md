#### Git拉取

* 从远程仓库克隆项目到本地仓库（第一次使用）：

  1. 进入工作空间，右键空白处点击Git Bash Here，打开Git工具

  2. 输入代码，克隆远程仓库

```git
git clone https://gitee.com/Guqier/SMP.git
```

  3. Git在工作空间创建名为SMP的目录，即为项目目录
  
  4. 输入`cd SMP`进入项目目录

* 从远程仓库更新项目资源到本地（第二次及以上）

  1. 进入SMP项目目录，右键空白处点击Git Bash Here，打开Git工具

  2. 输入代码，更新项目资源

```git
git pull
```

* 从本地仓库提交到远程仓库

```git
git add .
git commit -m [提交信息]
git push
```
